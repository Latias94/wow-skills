# 与装备路线技能的分工

## 使用本 skill

当用户问：

- 这件装备为什么强？
- 这个饰品怎么配爆发？
- 属性为什么这样堆？
- 这套天赋为什么影响装备收益？
- 我拿到这件以后循环是否改变？

本 skill 负责机制解释、打法调整和模拟结果解读。

## 转交 wow-voidforge-roll-planner

当用户问：

- 本周 R 币投哪个副本或 Boss？
- 宏伟宝库拿装备还是 6 代币？
- Voidforge bonus roll 池怎么规划？
- 未来 1-3 周追哪个主池？
- 领奖励或 roll 前切哪个拾取专精？

应使用 `wow-voidforge-roll-planner`。如果同时需要解释某件装备对职业的价值，可以先用本 skill 解释机制，再让 roll planner 做路线决策。

## 混合问题处理

推荐顺序：

1. 本 skill 判断装备/天赋/机制价值：它为什么适合这个专精。
2. `wow-voidforge-roll-planner` 判断获取路线：本周是否值得投入核心、宝库是否该拿。

不要把“这件装备强”直接等同于“本周一定该 roll 它”。
