# 回答骨架

## 入门教学

适用于“小号不会玩”“循环怎么打”“先告诉我大概怎么按”。

```text
一句话结论
当前默认假设
机制图：这个专精靠什么打伤害/治疗/抗压
简化循环：单体 / AOE / 爆发
为什么这么按：每个关键动作的机制原因
最常见 3 个错误
下一步训练目标
需要用户补充的信息
```

## 机制理解

适用于“为什么攻略这样写”“这个天赋改变了什么”。

```text
先给结论：攻略这么写的核心原因
技能/天赋文本拆解
因果链：A 触发/放大 B，所以 B 要放在 C 之前
场景边界：什么目标数、什么战斗时长、什么天赋下成立
常见误区
如何在木桩或日志里验证
```

## 天赋比较

适用于“这套天赋和那套哪个好”“我当前天赋怎么打”。

```text
结论：按玩家目标推荐哪套
玩家当前天赋改变点
主流构筑对比
循环变化表：天赋 | 改变的机制 | 打法影响
适用场景：团本/M+、单体/AOE、容错/上限
需要 /simc 或日志确认的点
```

## 日志复盘

适用于 WCL、低伤害、容易死、爆发打不出来。

```text
先看结论：最大问题 1-3 个
战斗背景：Boss/地下城、时长、死亡、装等、天赋
输出/治疗/坦克核心指标
机制原因：这些指标为什么会导致低表现
下一把只练 2-3 件事
如果需要，列出要进一步查看的日志页面
```

详细检查项读取 `log-review-rubric.md`。

## M+ 专项

适用于“大米怎么打”“某本怎么处理”“AOE 为什么低”。

```text
结论：这把最该优化的不是/是哪个循环点
怪包节奏：短包、长包、双目标、Boss、危险波次
爆发规划：哪些包交大招，哪些包保留
功能性：打断、控制、驱散、群控、减伤、位移
天赋调整：哪些点是伤害，哪些点是生存/工具
练习目标：下一把看哪些指标
```

## 装备/属性/饰品解释

适用于“这件为什么强”“属性怎么选”“饰品哪个好”。

```text
方向性结论
机制原因：它强化了什么窗口/资源/伤害类型
场景差异：单体、AOE、M+、团本、Boss
数据来源：个人 sim 优先，没有则说明使用通用资料
建议：是否需要 /simc、Top Gear、Droptimizer
```
