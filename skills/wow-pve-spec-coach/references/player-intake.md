# 玩家信息收集模板

## 最小输入

只要有这些信息，就可以给方向性教学：

- 职业与专精
- 主要玩法：M+ / 团本 / 双修
- 当前版本或服务器环境
- 玩家目标：小号入门、普通提升、冲层、开荒、日志提升

## 推荐输入

有这些信息，可以给更贴近角色的建议：

- 装等、套装件数、Hero talents
- 当前天赋导入字符串、Wowhead talent link 或截图
- 当前武器、双饰品、关键制作装
- 常打钥石层数、团本难度和 Boss 进度
- 一段 WCL 链接、Raidbots 链接或 `/simc`

## 精确输入

用于日志复盘或装备决策：

- Warcraft Logs 战斗链接，最好指定 Boss/地下城和哪一把
- `/simc` 文本或 Raidbots Top Gear / Droptimizer 链接
- 当前天赋导入字符串
- 用户认为自己困惑的点：爆发低、AOE 低、单体低、经常死、资源乱、按键太多

## 询问策略

不要一次问太多。优先问最能改变答案的问题：

- 循环问题：先要天赋导入字符串。
- 日志问题：先要 WCL 链接和目标战斗。
- 装备问题：先要 `/simc` 或 Raidbots。
- M+ 问题：先要层数、地下城、队伍职责和常见死亡/低伤场景。

如果用户只是想先学小号，直接给保守入门版，再在结尾请用户补天赋串。
