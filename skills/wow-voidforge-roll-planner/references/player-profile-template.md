# 玩家画像模板

开始规划前，优先收集以下信息。用户没给全时，不要卡住；可以先按保守假设继续，但必须写明假设。

## 必填

- 职业专精
- Hero talents
- 主要内容定位
  - 团本玩家
  - M+ 玩家
  - 双修玩家
- 当前主要玩法
  - 团本为主
  - M+ 为主
  - 两边都打
- 当前大致装等
- 当前可稳定完成的 M+ 层数
- 当前可稳定击杀的团本难度 / Boss 段位

## 强烈建议补充

- 当前 2 件套 / 4 件套进度
- 当前双饰品
- 当前主手 / 副手
- 当前最弱的 2 到 4 个槽位
- 当前宏伟宝库候选
- 当前已做的制作装
- 当前还有没有火花、制作预算、纹章预算

## 如果用户愿意给更精确信息

- `/simc` 导出的角色字符串
- 角色面板截图
- 宏伟宝库截图
- 当前背包里待选装备

## 输入完整度标记

- 只给玩法、装等、可打内容：`基础档`
- 再给当前双饰品、武器、套装、宝库：`进阶档`
- 再给 `/simc`：`精确档`

## 默认假设模板

如果信息缺失，可用下面这种写法：

- 默认按 `双修偏 M+`
- 默认按 `M+ 为主`
- 默认按 `本周稳定 +10`
- 默认按 `英雄团不稳定 / 史诗团未 farm`
- 默认按 `4 件套未完成`
- 默认按 `没有现成稀有团本 BIS`

不要把这些默认假设藏起来，必须在答案开头说清楚。
